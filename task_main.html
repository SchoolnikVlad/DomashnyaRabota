Задание: Текст каждого задания описывает сам себя. Это означает, что текст "Этот текст красный" должен быть красного цвета. Т.е. смысл текста должен соответствовать его внешнему виду.

В рамках задания необходимо найти ошибки в HTML или CSS, которые приводят к тому, что смысл текста не соответствует его внешнему виду.

В качестве результата задания необходимо прислать исправленные HTML- и CSS-файлы, чтобы смысл текста стал соответствовать его внешнему виду. При этом изменения нужно вносить во внешний вид текста, а не в его смысл!

Желательно вносить минимальное число изменений, т.е., например, не переписывать полностью css-стили

Если для выполнения задания можно обойтись без инлайнового стиля, то нужно обойтись без него

Пример №1:

<span class="red">Этот текст зелёный</span>

Смысл текста не отражает его внешний вид Проблема в неверном значении атрибута class. Должно быть

<span class="green">Этот текст зелёный</span>

Пример №2:

<div class="green">Этот текст зелёный</div>

При этом в css указано

div.green {
  color: red;
}

Смысл текста не отражает его внешний вид Проблема в неверном значении атрибута color. Должно быть

div.green {
  color: green;
}
